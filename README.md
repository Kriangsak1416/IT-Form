# IT-Form

แบบฟอร์มศูนย์คอมพิวเตอร์ โรงพยาบาลแพร่

### Prerequisites

- Node.js 18.x or later
- npm or yarn

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Docker Development
```bash
docker compose up --build
```

## Technologies Used

- Next.js 16
- TypeScript
- Tailwind CSS v4
- ESLint
- Docker
